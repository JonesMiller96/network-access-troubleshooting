# Network Access Troubleshooting Guide

This repository documents common methods to resolve access issues caused by ISP-level restrictions or DNS filtering.

## Methods

### 1. DNS Switching

Using public DNS providers can sometimes bypass basic filtering:

- Google DNS: 8.8.8.8 / 8.8.4.4  
- Cloudflare DNS: 1.1.1.1  

### 2. Private DNS (Android)

You can configure Private DNS:

dns.google

### 3. Browser-based Solutions

Some browsers include built-in VPN or proxy features:
- Opera (built-in VPN)
- Extensions like Browsec

### 4. Domain Changes

Some services may become inaccessible not because they are offline, but because their domain has changed.

Tracking updated domains can help restore access more quickly.

## Notes

- Not all methods work in every region  
- Combining DNS + alternative access methods is often more effective
